# monkey-ui
基于react的前端框架，参考antd


##install:
<code>npm install monkeyui</code>

##build:
<code> npm install </code>

<code> gulp build</code>	

##dev run:
<code> npm run dev </code>

<code>	http://localhost:8080/index.html</code>


## use monkeyui 

<code>import MonkeyUi from 'monkeyui'</code>
